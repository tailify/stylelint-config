> Tailify's very own opinionated stylelint configuration.

## Installation

```
$ npm install --save-dev @tailify/stylelint-config
```

## Usage

To get started, add this to your `.stylelintrc.js` file:

```js
module.exports = {
  extends: ['@tailify/stylelint-config']
};
```
